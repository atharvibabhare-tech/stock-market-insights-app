# 📈 Stock Market Insights Dashboard

An interactive Stock Market Insights Dashboard built using **Streamlit**, **Pandas**, and **Matplotlib**.

This app provides visual analysis of Nifty stock data with filtering, trends, and interactive charts.

---

## 🚀 Features

- 📊 Interactive stock data visualization
- 📈 Trend analysis using line charts
- 📅 Date-based filtering
- 📂 Multi-page Streamlit app structure
- 📉 CSV-based stock dataset integration

---

## 🛠 Tech Stack

- Python
- Streamlit
- Pandas
- Matplotlib
